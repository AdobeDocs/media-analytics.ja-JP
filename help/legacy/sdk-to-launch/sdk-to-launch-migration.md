# スタンドアロンの Media SDK から Adobe Launch への移行

>[!NOTE]
>Adobe Experience Platform Launch は、Experience Platform のデータ収集テクノロジースイートとしてリブランドされています。 その結果、製品ドキュメント全体でいくつかの用語の変更がロールアウトされました。 用語の変更点の一覧については、次の[ドキュメント](https://experienceleague.adobe.com/docs/experience-platform/tags/term-updates.html?lang=ja)を参照してください。

このドキュメントでは、主にMedia Analyticsの実装の違いについて説明します
スタンドアロンのMedia SDKとMedia Analytics Launch拡張機能の間の接続です。

Adobe Cloud Platform Launchは、あらゆる分析をシンプルな方法でデプロイ、管理でき，
関連性の高い顧客を強化するために必要なマーケティング、広告ソリューション
コンバージョン指標を考慮します。 Adobe Cloud Platform Launchは誰でもアプリケーションを
独自の統合を行っています。 これらの統合は拡張機能と呼ばれます。
拡張機能は、Launchの顧客がアプリストア体験で利用できるため、マーケターは
迅速にソリューションをインストール、設定、展開することができます。

Adobe Media Analytics for Audio and Video は、Web とモバイルの両方で Launch 拡張機能として使用できます。

* [Web Launchのドキュメント](https://experienceleague.adobe.com/docs/experience-platform/tags/extensions/adobe/media-analytics/overview.html?lang=ja)
* [モバイル版ローンチのドキュメント](https://developer.adobe.com/client-sdks/documentation/adobe-media-analytics/)
